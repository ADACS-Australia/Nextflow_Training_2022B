NextFlow

NextFlow enables scalable and reproducible scientific workflows using software containers.

  • Windows
  • MacOS
  • Linux
  1. Enable the Windows Subsystem for Linux
  2. Enable the Virtual Machine Feature
  3. Download the Linux Kernel Update Package
  4. Set WSL2 as your default version
    5. Full instructions from the Nextflow site. (Includes install instructions for Docker which will be useful).
  5. You will also need to have Java installed to run NextFlow, see this Java Install guide for instructions for Windows.
  1. From a terminal window run the following wget -qO- https://get.nextflow.io | bash or if you prefer curl run curl -s https://get.nextflow.io | bash. This will create the nextflow main executable file in the current directory.
  2. Make the binary executable on your system by running chmod +x nextflow.
  3. Move the nextflow file to a directory accessible by your $PATH variable
  4. You will also need to have Java installed to run NextFlow, see this Java Install guide for instructions for MacOS.
